# Quick Checklist: Directories to Personalize

## ✅ Must Change (Core Personalization)

- [ ] **_config.yml** - Update site title, name, bio, URLs, social media links
- [ ] **_data/navigation.yml** - Configure top menu items
- [ ] **_pages/about.md** - Write your personal introduction
- [ ] **images/profile.png** - Replace with your profile picture
- [ ] **_publications/** - Add your publications (delete sample files)
- [ ] **_talks/** - Add your talks/presentations (delete sample files)
- [ ] **_teaching/** - Add your teaching experience (delete sample files)
- [ ] **_posts/** - Add blog posts or remove directory if not blogging
- [ ] **_portfolio/** - Add portfolio items or remove if not needed

## 📝 Should Change (Content)

- [ ] **_pages/cv.md** - Update your CV
- [ ] **files/** - Upload your actual PDFs (papers, slides, etc.)
- [ ] **_data/cv.json** - If using JSON CV format

## 🎨 Optional (Customization)

- [ ] **_sass/** - Customize visual appearance
- [ ] **_includes/footer.html** - Modify footer content
- [ ] **_layouts/** - Change page layouts
- [ ] **assets/** - Advanced styling/functionality

## 🔧 Utility (Keep or Use)

- [ ] **markdown_generator/** - Use to bulk-create pages from CSV
- [ ] **talkmap/** - Generate map of talk locations (optional)

## Summary of Key Directories

**Content Collections:**
- `_publications/` - Academic publications
- `_talks/` - Presentations and talks
- `_teaching/` - Teaching assignments
- `_posts/` - Blog posts
- `_portfolio/` - Portfolio projects
- `_pages/` - Individual pages

**Configuration:**
- `_config.yml` - Main config
- `_data/` - Navigation, UI text, CV data

**Static Resources:**
- `files/` - PDFs and documents
- `images/` - Images and profile picture

**Theme/Styling (Advanced):**
- `_sass/` - Stylesheets
- `_includes/` - HTML partials
- `_layouts/` - Page templates
- `assets/` - Compiled assets
